# Glossary

- Seed: stable entry point in memory.
- Balloon: local expandable scope around one or more seeds.
- Bridge: typed relation between seeds/records.
- Evidence: auditable observed fact or event.
- Concept: generalized meaning promoted from evidence.
- Route: traversable path that worked for a task/query/repair.
- Source seed: external dataset/source reference.
- Fragment ref: pointer to a bounded external data fragment.
- Hydrated fragment: external fragment copied into BalloonDB cache with provenance.
- WAL: append-only write-ahead log for recovery.
- BQL: Balloon Query Language.
