# Security Policy

BalloonDB is currently experimental software.

## No warranty

The software is provided as-is, without warranty of any kind. Users run it at their own risk.

## No security SLA

This is not a commercial service and no response-time guarantee is provided.

## Recommended use

Do not use BalloonDB for production, regulated, sensitive, medical, financial, legal, or safety-critical data without your own independent audit, threat model, backups, and validation.

## Reporting issues

Use GitHub Issues for public bug reports. Do not post secrets, API keys, private datasets, personal data, or production credentials in issues.
